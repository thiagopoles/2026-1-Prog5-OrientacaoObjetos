Questão 3: Complete o código do método depositar que adiciona um valor ao saldo da conta.
public void depositar(double valor) {
 	this.saldo = this.saldo + valor;
}


Questão 4: Complete o código do método sacar que subtrai um valor do saldo da conta, se o saldo for
suficiente.
public void sacar(double valor) {
 	if(this.saldo >= valor) {
 		this.saldo = this.saldo - valor;
 	}
}
